# 🚗  Full Stack Ride Booking Platform

This repository contains a **full-stack Uber clone** built to replicate the core functionality of a modern ride-hailing platform.  
The project is designed with a **clean separation of frontend and backend**, following real-world production architecture.

---

## 📌 Project Overview

The application allows users to:
- Sign up and log in securely
- Book rides in real time
- Track ride status
- Interact with drivers
- Experience smooth client-side navigation

The system is divided into two independent layers:

- **Frontend** → User interface and client-side logic  
- **Backend** → Authentication, APIs, database, and business logic  

Each layer has its **own README** with setup instructions and technical details.
